Git ve GitHub Temelleri

Git, yazılım geliştiriciler için vazgeçilmez bir versiyon kontrol sistemidir. Bu yazıda temel komutları ve GitHub üzerinde repo yönetimini ele alıyoruz.

  1. git init – Yeni bir repo oluşturur.
  2. git commit -m – Değişiklikleri kaydeder.
  3. git push – Değişiklikleri uzak repoya gönderir.

Daha fazla bilgi için Git Resmi Dokümantasyonu ve GitHub sitelerini ziyaret edebilirsin.
